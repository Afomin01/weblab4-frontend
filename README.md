# weblab4-frontend
Web programming lab 4 frontend made at ITMO University by Artem Bakin and Dmitry Kupershtein.

## Task
Develop a frontend on React + Redux stack for backend RESP API.

## Components structure
```
weblab4-frontend
├── welcome page
│   └── AuthorizationForm
│       ├── LoginForm
│       └── SignUpForm
└── main page
    ├── UserModule
    ├── PointsInputForm
    └── MainPageEntryTable
```
